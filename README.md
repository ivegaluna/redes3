## Publicación en GitHub Pages

1. Crear un nuevo repositorio en GitHub (por ejemplo, `redes-uam-26i`).
2. Subir todos los archivos del proyecto.
3. Activar GitHub Pages desde la rama principal (main) y carpeta raíz (/).
4. Esperar unos minutos y acceder a tu sitio en  
   `https://<usuario>.github.io/redes-uam-26i/`.
